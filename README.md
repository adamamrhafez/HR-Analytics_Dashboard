# HR Analytics Dashboard

This project is a Power BI dashboard focused on analyzing Human Resources data to provide actionable insights.

## Project Overview
The main goal of this dashboard is to track key HR metrics, such as employee turnover, recruitment efficiency, and department performance.

## Tools Used
- Power BI
- Excel

## Features
- Interactive visualizations.
- Key Performance Indicators (KPIs) for HR management.
- Dynamic filtering to explore data across different departments.
